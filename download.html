预编译的文件

你可以直接下载预编译的文件，其中包含了编译并压缩过的 CSS 和 JavaScript 文件，以及 Roboto 字体文件和 Material Icons 图标文件。

下载 MDUI

源码文件

MDUI 的所有源码都托管在 Github 上，你可以从中了解 MDUI 的实现方式，或帮助修复存在的 bug。或进行二次开发，并自行编译 MDUI。

访问 Github

使用 CDN

使用 CDN 是我们最为推荐的方式，因为它在很多时候可以直接从缓存中加载，拥有比其他方式更快的加载速度。 

<link rel="stylesheet" href="https://unpkg.com/mdui@1.0.2/dist/css/mdui.min.css">
<script src="https://unpkg.com/mdui@1.0.2/dist/js/mdui.min.js"></script>

通过包管理器安装

通过包管理器，你可以将 MDUI 的源文件和编译后的文件加入到你的已有项目中。

npm 

npm install mdui --save

yarn 

yarn add mdui
